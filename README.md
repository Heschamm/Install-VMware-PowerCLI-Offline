# PowerCLI Offline Installer with File Browser
PowerShell script to select an offline bundle for VMware PowerCLI and install it offline. No Internet Access Needed!
VMware PowerCLI download link: https://developer.broadcom.com/tools/vmware-powercli/latest/ 
Use Windows Explorer to navigate to the script, right-click and select: Run with PowerShell - Follow the steps
Use following powershell command to check a successfull installation: $ Get-Module -ListAvailable -Name VMware.PowerCLI
VMware by Broadcom official PowerCLI documentation to be found in the following link: https://techdocs.broadcom.com/content/dam/broadcom/techdocs/us/en/pdf/vmware/vcf/power-cli/latest/powercli.pdf
